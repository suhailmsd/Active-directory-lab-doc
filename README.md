## 1️⃣ Introduction – What is Active Directory?

Active Directory (AD) is a directory service developed by Microsoft that enables centralized
management of users, computers, and security policies within a Windows domain environment.

It allows:
• User authentication
• Authorization
• Centralized password management
• Device management
• Domain-based login
